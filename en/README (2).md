# Skills for the English eBook

OpenClaw skills referenced in chapters 11–15 of the upcoming English edition of *"OpenClaw for SMBs"*.

## Status

Work in progress. English translations of the skills will be added here as the English edition progresses.

## Structure

The English skills will mirror the German structure:

```
en/skills/
└── lead-tracker/
    ├── SKILL.md
    ├── references/
    └── data/
```

Each skill is functionally identical to its German counterpart under `de/skills/` — only the natural-language guidance inside `SKILL.md` and the `references/` files is translated. The CSV schema and file paths remain the same across languages.

## Using the German skills in an English setup

The German skills work technically even in an English-language OpenClaw configuration. The agent will simply reply in German when the skill is active. If that is acceptable for your setup, you can use `de/skills/` directly.
