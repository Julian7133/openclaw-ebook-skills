# OpenClaw eBook Skills

Begleitrepository zum Buch *„OpenClaw für den Mittelstand"* von Julian Berlow.

## Inhalt

- `de/setup/KAPITEL-03.md` bis `de/setup/KAPITEL-10.md`
- `de/workflows/KAPITEL-11.md` bis `de/workflows/KAPITEL-15.md`
- `de/skills/lead-tracker/`
- `de/skills/daily-briefing/`
- `de/skills/market-radar/`

## Skill kopieren

```bash
cd ~/.openclaw/workspace
git clone https://github.com/Julian7133/openclaw-ebook-skills.git begleit-repo
cp -r begleit-repo/de/skills/<skill-name> skills/
```
